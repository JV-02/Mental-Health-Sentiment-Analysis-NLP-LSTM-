# Mental Health Sentiment Analysis using NLP & Deep Learning
Sentiment classification of mental health text using NLP preprocessing and LSTM deep learning model.

## Objective
Classify text into sentiment categories using Natural Language Processing and LSTM neural networks.

## Tech Stack
Python, NLTK, TensorFlow/Keras, Scikit-learn

## Workflow
- Text cleaning and preprocessing
- Tokenization and sequence padding
- Label encoding
- LSTM model training
- Performance evaluation

## Key Learnings
- NLP preprocessing techniques
- Sequence modeling
- Deep learning for text classification

## Files
- Mental_Health_Sentiment_Analysis.ipynb

## How to Run
Install required libraries and run the notebook.
